Text And Fonts

Text有一些属性，我们在开发中运用很多，我们也来聊聊它们，以加深对它们的理解，作为网页内容的主体之一，文字 的处理占据开发一部分，文字的格式化需要熟练。我们下来说说：

Text Color

文本颜色，顾名思义，主要是对文字颜色的格式化，属性名为color,而它的值一般有几种

  • HEX:#ff0000
  • RGB:rgb(255,0,0)
  • color name:red

See the Pen QyNMLx by Tab (@ivanberry) on CodePen.

对齐

关于对齐这个属性，就居中而言就可以上演一场大戏，因为是将文字的格式话，我们就不展开，就说text-algin 它用于文字的水平对齐方式，文字可以左对齐，右对齐，居中或者两端对齐.闲话不扯，属性比较简单，直接上例子更好理解,

  • text-align: left;
  • text-align: right:
  • text-align: center;
  • text-align: justify;

See the Pen GoZvjM by Tab (@ivanberry) on CodePen.

Text Decoration

这个属性值也好理解，更改或者移除一些文字的样式，多用于链接样式，看例子，看例子

See the Pen pgyrjq by Tab (@ivanberry) on CodePen.

Text Transformation

text-transform可用于改变文字的大小写，汉字里面好像没有很多用处啊，不是很实用啊

See the Pen WrwERa by Tab (@ivanberry) on CodePen.

Text Indentation

缩进，对于传统文章中，尤其是中文编辑中出现的较多的属性，主要更改首行的缩进。

See the Pen qbZXmN by Tab (@ivanberry) on CodePen.

字母间距，字间距，行高，文字方向

这些用的相对少一些，放在一起讲了，主要是对文字的细节调整

  • 字母间距：letter-spacing
  • 字间距：word-spacing
  • 行高：line-height
  • 文字方向：direction

See the Pen VeazWb by Tab (@ivanberry) on CodePen.

See the Pen vLLOJw by Tab (@ivanberry) on CodePen.

Fonts

字体，也是很常用的一个属性，font包含有好几个属性值，以下一一列出讲解，不是很难，简单 说一下，要想参照更加细节的用法，可以去W3C找标准读一读。

  • font-family:定义字体
  • font-style:字体样式
  • font-weight:字体厚实度
  • font-size: 字体大小
  • font-variant:是否大写

用一个例子来说明吧，这个应该是不难掌握我的，但是细节的实现就必须详细了解了，比如字体名存有空格的必需放在引号内 ，在使用特定字体时，在字体最后备份个常见字体，实现“优雅后退”等等，需要在实践多多总结。

See the Pen XXdaQK by Tab (@ivanberry) on CodePen.

关于字体的属性可由一个值来表示font，对它没有太多的限制，跟它的子属性一般用法，但有一点需要特别注意 ，font值中必须要有字体和字体大小,而且它们的顺序必须是字体大小在钱，字体在后，不然会造成声明失效，更有可能导致以下的 所有CSS申明失效，还有一点提及，可以在字体属性值后/行高来增添行高值。例子尽量描述出问题所在，最好的实践方法就是去试试。

See the Pen VeaMZQ by Tab (@ivanberry) on CodePen.
